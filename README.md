# Trade room

Присутствуют два бота в комнате (для примера). Чтобы добавить больше одного пользователя, нужно открыть больше одной вкладки.

Стек:
Frontend:
React

Backend:
nginx + node js

Ипользовалась socket.io для общения между участниками и сервером.
